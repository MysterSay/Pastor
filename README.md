# Підтримка поруч

Динамічний вебсайт для підтримки людей, про яких ми піклуємось, і для спонсорства тих, хто віддає свій час служінню іншим.

Сайт автоматично підтягує анкети з бази Notion через Cloudflare Worker, відображає їх у вигляді адаптивних плиток і відкриває вибрану анкету в окремому модальному вікні.

---

## Основні можливості

- динамічне завантаження анкет з Notion
- фільтрація анкет за статусом публікації
- адаптивна сітка плиток (9:16)
- випадковий вибір анкети
- пошук по імені, церкві, місцю служіння, нотатці або історії
- модальне вікно анкети з розмиттям фону
- сучасний анімований фон
- фронтенд на HTML, CSS, JavaScript
- бекенд через Cloudflare Worker

---

## Технології
